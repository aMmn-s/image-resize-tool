# Vercel にデプロイする手順（ターミナルで1つずつ）

専門用語をなるべく使わず、コマンドを1つずつ説明します。

---

## 準備：ターミナルを開く

- Cursor では **メニュー → ターミナル → 新しいターミナル** で開けます。
- またはキーボードで **Ctrl + `**（バッククォート）でも開けます。

開くと、黒い画面に文字が並んでいる状態になります。ここに「コマンド」を打ち込みます。

---

## ステップ1：プロジェクトのフォルダに入る

**打ち込むコマンド：**

```bash
cd /Users/tsukasa/Desktop/AI用/Cursor/image-resize-tool
```

**Enter キーを押す。**

**何をしているか：**  
`cd` は「このフォルダに移動する」という意味です。  
画像リサイズアプリのファイルが入っているフォルダに、今から操作する場所を移しています。

---

## ステップ2：Vercel のツール（CLI）を一度だけ用意する

**打ち込むコマンド：**

```bash
npx vercel
```

**Enter キーを押す。**

**何をしているか：**  
`npx` は「指定したツールをその場で使う」ための仕組みです。  
`vercel` は Vercel にアプリを送るためのツール（CLI）の名前です。  
初回は「ダウンロードしています」のような表示が出ることがあります。そのあと、質問がいくつか出ます。

---

## ステップ3：質問に答える（初回だけ）

コマンドを打つと、下のように **質問** が表示されます。順番に答えます。

### 質問1：「Set up and deploy?」

```
? Set up and deploy "~/Desktop/AI用/Cursor/image-resize-tool"? [Y/n]
```

**やること：** キーボードで **Y** を押して **Enter**。

**意味：** 「このフォルダを Vercel にデプロイする準備をして、送りますか？」という確認です。Y = はい。

---

### 質問2：「Which scope do you want to deploy to?」

```
? Which scope do you want to deploy to?
  > あなたの名前 (Personal)
```

**やること：** 矢印キーで **自分の名前（Personal）** を選んで **Enter**。

**意味：** 「どのアカウント（誰の名前）でデプロイしますか？」です。個人なら自分の名前を選びます。

---

### 質問3：「Link to existing project?」

```
? Link to existing project? [y/N]
```

**やること：** 何も押さずに **Enter**（N のまま）。

**意味：** 「すでに Vercel にあるプロジェクトに紐づけますか？」です。初めてなら「いいえ」なので N のまま Enter でよいです。

---

### 質問4：「What’s your project’s name?」

```
? What's your project's name? image-resize-tool
```

**やること：** そのまま **Enter**。

**意味：** 「プロジェクトの名前は？」です。かっこ内の名前で問題ないので、そのままで大丈夫です。

---

### 質問5：「In which directory is your code located?」

```
? In which directory is your code located? ./
```

**やること：** そのまま **Enter**。

**意味：** 「どのフォルダにプログラムのコードがありますか？」です。`./` は「今いるこのフォルダ」という意味なので、そのままでよいです。

---

## ステップ4：しばらく待つ

ここまで答えると、Vercel が **ビルド（プログラムを本番用にまとめる作業）** を始めます。

- 「Building...」や「Uploading...」のような表示になります。
- 数十秒〜1分ほどかかることがあります。

---

## ステップ5：URL が表示されたら完了

最後に、例えば次のような表示が出ます。

```
✅  Production: https://image-resize-tool-xxxx.vercel.app
```

**やること：**  
この **https://…** の部分をコピーして、ブラウザのアドレス欄に貼り付けて開く。

**意味：**  
これが「あなたのアプリがインターネット上で見られるアドレス」です。ここで画像リサイズツールが動いています。

---

## 本番用（正式な公開）にしたいとき

上までで「プレビュー用」の URL が出ています。  
「本番（本式の公開）」にもしたい場合は、もう一度ターミナルで次を打ちます。

**打ち込むコマンド：**

```bash
npx vercel --prod
```

**Enter キーを押す。**

**何をしているか：**  
`--prod` は「本番用にデプロイする」という指定です。  
質問は出ず、そのままビルドが始まり、本番用の URL が表示されます。  
この URL が「正式に公開されているアドレス」になります。

---

## まとめ（コマンドの順番だけ）

1. `cd /Users/tsukasa/Desktop/AI用/Cursor/image-resize-tool`  → Enter
2. `npx vercel`  → Enter
3. 質問には Y → 自分の名前 → Enter（Nのまま）→ Enter → Enter
4. 表示された URL をブラウザで開く
5. 本番にしたいときは `npx vercel --prod`  → Enter

何かエラーや違う表示が出たら、その画面の内容を教えてもらえれば、次の操作を案内できます。
